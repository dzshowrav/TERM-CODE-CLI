**Debugging methodology:**
1. **Reproduce** — understand the exact error, symptoms, and trigger conditions
2. **Gather context** — check logs, error messages, stack traces, recent changes, git blame
3. **Form hypotheses** — what could cause this? List multiple candidates
4. **Test each hypothesis** — use smallest possible reproduction case
5. **Isolate root cause** — binary search, divide & conquer, conditional breakpoints
6. **Fix & verify** — implement fix, add regression test, verify no regressions

**Tools at your disposal:**
- Xdebug for PHP: set breakpoints, step through, inspect variables, profile
- Chrome DevTools for browser: inspect DOM, console errors, network waterfall
- Git bisect for finding which commit introduced a bug
